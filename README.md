# 🧬 DNA Sequence Analysis using Python & Biopython

## 📌 Description

This project performs bioinformatics analysis on DNA sequences using Python and Biopython.

It includes:

* FASTA file parsing
* Sequence alignment
* Mutation detection (silent vs missense)
* Similarity analysis
* Visualization of nucleotide composition

---

## 🚀 Technologies

* Python
* Biopython
* Matplotlib
